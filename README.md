This repository holds the source code and documentation for the Final Degree Project (TFG) focused on developing a 
**comprehensive Point of Sale (POS) System** specifically designed for the efficient management of restaurants, cafes, or fast-food chains.

The main objective is to build a robust, modular, and user-friendly application, covering both the user interface (Front-end) and the business logic and data persistence (Back-end) through a Full-Stack development methodology.

The project is being developed using a modern technology stack, adhering to a Full-Stack development architecture.
